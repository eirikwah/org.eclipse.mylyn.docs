h1. Generate EPUB from markup

p. Lightweight wikitext markup can easily be converted into EPUBs. Right click on any supported markup file (ending in @*.textile@, @*.mediawiki@ etc), then select **Wikitext > Generate EPUB** from the context menu. This will bring up the dialog shown below.

!generate-epub.png! 

As the EPUB specification requires that certain properties are set for a publication - this dialog will offer the user interface for setting these. At a minimum you will have to enter a **title**, **language** and **subject**.
